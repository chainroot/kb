# Index of Cosmos Network Modules

This document serves as an index for the comprehensive documentation of each module in the Cosmos network. Click on a module name to access its detailed documentation.

## Cosmos Network Modules

1. [Staking Module](./staking-module.mdx) - Manages staking and delegation processes.
2. [Governance Module](./governance-module.mdx) - Facilitates on-chain governance and voting.
3. [Bank Module](./bank-module.mdx) - Handles token transfers and account balances.
4. [Slashing Module](./slashing-module.mdx) - Implements validator punishment mechanisms.
5. [Distribution Module](./distribution-module.mdx) - Manages fee and reward distribution.
6. [Auth Module](./auth-module.mdx) - Manages authentication of accounts and transactions.
7. [Authz Module](./authz-module.mdx) - Enables granular permissions for accounts.
8. [Mint Module](./mint-module.mdx) - Manages the creation of new tokens and inflation.
9. [Crisis Module](./crisis-module.mdx) - Handles network emergencies and halts.
10. [IBC Module](./ibc-module.mdx) - Facilitates inter-blockchain communication.
11. [Params Module](./params-module.mdx) - Manages network parameters.
12. [Evidence Module](./evidence-module.mdx) - Handles evidence for validator misbehaviors.
13. [Upgrade Module](./upgrade-module.mdx) - Coordinates network upgrades.
14. [Feegrant Module](./feegrant-module.mdx) - Manages fee payment delegations.
15. [NFT Module](./nft-module.mdx) - Supports Non-Fungible Token functionalities.
16. [CosmWasm Module](./cosmWasm-module.mdx) - Enables smart contracts in the network.
17. [EVM Module](./evm-module.mdx) - Supports Ethereum Virtual Machine compatibility.
18. [Protocolpool Module](./protocolpool-module.mdx) - Manages community pool funds.
19. [Circuit Module](./circuit-module.mdx) - Implements a circuit breaker mechanism.
20. [Genutil Module](./genutil-module.mdx) - Provides genesis utilities.
21. [Consensus Module](./consensus-module.mdx) - Manages consensus parameters.
